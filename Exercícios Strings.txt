OK - 1. Escrever uma função C que recebe uma string e transforma todas as letras dessa string em maiúsculas. OK!!!

OK - 2. Escrever uma função C que recebe uma string e altera cada palavra nessa string para que a primeira letra seja maiúscula e as demais minúsculas. OK!!!

QUASE OK - 6. Escrever uma função que recebe uma matriz de strings e a preenche com nomes de pessoas aleatórios. Um nome deve ser composto por um primeiro nome e dois sobrenomes.

QUASE OK - 7. Faça uma função que recebe um nome completo e mostre a uma string formada pela primeira letra de cada palavra do nome. Não se devem mostrar a primeira letra de preposições como: do, de, etc. A abreviatura deve vir separada por pontos. Ex: Paulo Jose de Almeida Prado. Abreviatura: P.J.A.P. 

8. Fazer uma função para que recebe uma string e retorna o número de ditongos ou hiatos nessa string.
